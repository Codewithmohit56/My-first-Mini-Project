# My first-Mini-Project
🍔 Online FoodShop – Mini Web Project

An interactive food ordering web application built using HTML, CSS, and JavaScript.
This project simulates a simple online food delivery platform where users can browse restaurants, explore menus, add items to a cart, and place orders with delivery details.

It demonstrates the fundamentals of front-end development, including responsive UI design, DOM manipulation, cart management, and form handling.

📌 Project Overview

Online FoodShop is designed to replicate a basic workflow of a modern food delivery website.
Users can:

⦿ Browse different restaurants

⦿ View a dynamic menu

⦿ Filter food by category

⦿ Add food to a shopping cart

⦿ Adjust quantity of items

⦿ View order summary

⦿ Enter delivery details

⦿ Select payment method (Cash / UPI)

This project provides hands-on experience with front-end development concepts while reinforcing core JavaScript skills.

⚙️ How It Works
1️⃣ Browse Restaurants

Users select a restaurant to see its menu.

2️⃣ Explore Menu

• Food items are dynamically loaded based on:

• Selected restaurant

Selected category filter

3️⃣ Add to Cart

Click ORDER NOW to add items.

Cart allows:

• Quantity adjustment

• Item removal

• Total calculation

4️⃣ Checkout

Users enter delivery details and choose payment method.

5️⃣ Order Confirmation

A unique Order ID is generated when the order is placed.
